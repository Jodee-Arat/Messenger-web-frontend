# WEB Manual Test Run Sheet

Рабочая таблица для ручного прогона web-версии.

Подробные шаги и пояснения смотри в:

- [WEB_MANUAL_TEST_CHECKLIST.md](./WEB_MANUAL_TEST_CHECKLIST.md)

## Как заполнять

- `Статус`: `OK`, `FAIL`, `SKIPPED`, `RETEST`
- `Факт`: что реально произошло при проверке
- `Комментарий`: детали, условия, аккаунты, URL, браузер
- `Баг`: номер бага, ссылка на задачу или краткое имя проблемы
